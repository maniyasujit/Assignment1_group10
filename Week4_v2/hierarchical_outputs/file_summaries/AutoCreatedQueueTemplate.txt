**Key Functionality:**
This Java class, `AutoCreatedQueueTemplate`, manages template settings for automatically created queues in a YARN (Yet Another Resource Negotiator) resource management system, specifically for the Capacity Scheduler.

**Core Logic:**
The class stores and applies template configuration properties for auto-created queues, following a precedence hierarchy (explicit > wildcard). It differentiates between common, leaf-specific, and parent-specific template properties when setting configurations for child queues.

**Inputs/Outputs:**

* **Inputs:**
	+ `CapacitySchedulerConfiguration` object
	+ `QueuePath` objects (for parent and child queues)
* **Outputs:**
	+ Populated template property maps (common, leaf-only, parent-only)
	+ Modified `CapacitySchedulerConfiguration` object with applied template settings for child queues
* **State Changes:**
	+ Updates to `CapacitySchedulerConfiguration` object
* **Side Effects:**
	+ None explicit, but influences queue configuration in the YARN resource management system
