This file implements the `UsersManager` class, which is responsible for managing users and their resource usage within a YARN (Yet Another Resource Negotiator) queue. It tracks user-specific data structures, computes user limits, and updates resource usage.

**Core Logic:**

1. **User Management**: Adds, removes, and retrieves users, maintaining a map of users with their respective resource usage and application details.
2. **User Limit Computation**: Calculates user limits based on queue capacity, user weights, and resource usage, considering both active and non-active users.
3. **Resource Usage Updates**: Synchronizes user resource usage during container allocation and release, updating usage ratios and triggering user limit recomputations as needed.
